Acoustic Passive Scan

Continuous frames of 1.5s call segments are parsed using speaker-independent Logistic Regression classification algorithms.

  • Timestep resolution: 1.5s windows
  • Feature vector: 120-dim MFCCs
  • Classifier: Logistic Regression
  • EER baseline: 5.92%

Dynamic Vocal Challenge

Active challenge-responses require spontaneous vocal replies, analyzed with automated offline speech-to-text content matches.

  • Engine: OpenAI Whisper (Tiny)
  • Device: CPU (Offline)
  • Content check: Replay Guard Filter
  • Adherence metrics: Levenshtein Match

Score Fusion Fuser

  • Active weight: Calibrated
